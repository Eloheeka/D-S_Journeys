
  # Travel Company Website Design

  This is a code bundle for Travel Company Website Design. The original project is available at https://www.figma.com/design/u8OPY3Ckhm2Jh2PlbWO76g/Travel-Company-Website-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Create a `.env.local` file and add:

  `VITE_WEB3FORMS_ACCESS_KEY=YOUR_WEB3FORMS_ACCESS_KEY`

  You can get the access key from Web3Forms and connect it to the inbox you want to receive submissions in, for example `m.eloheeka12@gmail.com`.

  Run `npm run dev` to start the development server.
